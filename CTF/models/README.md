# Trained Model Weights

This directory stores trained model weights generated by the training scripts.

## Files (generated after running training scripts)

| File | Model | Test MAE | Size (approx) |
|------|-------|----------|---------------|
| `cnn_lstm_best.keras` | CNN-LSTM ★ best | 0.0944°C | ~4 MB |
| `gru_best.keras` | GRU | 0.1074°C | ~5 MB |
| `lstm_best.keras` | LSTM | 0.1335°C | ~5 MB |
| `bilstm_best.keras` | Bi-LSTM | 0.1313°C | ~6 MB |
| `transformer_best.keras` | Transformer | 0.1146°C | ~3 MB |
| `lightgbm_model.pkl` | LightGBM | 0.1018°C | ~2 MB |
| `xgboost_model.pkl` | XGBoost | 0.1262°C | ~8 MB |
| `random_forest_model.pkl` | Random Forest | 0.1216°C | ~45 MB |
| `ridge_model.pkl` | Ridge* | 0.0064°C* | <1 MB |

*Ridge excluded from ensemble (linear trend extrapolation artefact).

## How to generate

```bash
# Train best model
python src/train_cnn_lstm.py

# Train Transformer
python src/train_transformer.py

# Train ML baselines (Ridge, RF, XGBoost, LightGBM)
python src/train_baselines.py
```

## Pre-trained weights

Pre-trained model weights are archived on Zenodo:
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.XXXXXXX.svg)](https://doi.org/10.5281/zenodo.XXXXXXX)

Download and place `.keras` and `.pkl` files in this directory.

## Git LFS

Model files are tracked via Git LFS (`.keras` ≥ 1 MB, `.pkl` ≥ 1 MB).
Run `git lfs pull` after cloning to download weights.
